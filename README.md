# Keyboard Launcher

A minimal Android launcher focused on **speed, focus, and simplicity**.

## Features
- App launcher with search
- Focus timer (Pomodoro-style)
- Notes page
- Deadlines & schedule view
- Swipe-based navigation

## Philosophy
One screen.
No clutter.
Keyboard-first.
Focus-driven.
